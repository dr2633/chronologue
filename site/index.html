Cursor Calendar

Cursor Calendar is a memory-grounded calendar interface that enables scheduling, summarization, and intelligent agent orchestration based on user-defined memory traces.

This project demonstrates how structured conversation data (e.g. goals, reflections, events) can be converted into actionable `.ics` calendar files and visualized through standard calendar clients.
